## Installation

Install Personal-Notes-App with npm

1. Clone the repo
   ```sh
   https://github.com/febrian2108/Personal-Notes-App.git
   ```
2. Go to project
   ```sh
   cd Personal-Notes-App
   ```
3. install npm 
   ```sh
   npm i
   ```
4. run project
   ```sh
   npm start
   ```
